# Installation

Prerequisites
- Node.js >= 18.17
- npm or pnpm

Install
1. Clone the repository and install deps: `npm install`
2. Copy `.env.example` to `.env` and adjust values.
3. Build Node and Worker bundles: `npm run build`

Run (Node)
- Dev: `npm run dev` or `npm run dev:watch`
- Prod: `npm run start:prod`
